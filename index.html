功能一

个股深度分析

输入股票代码，一键生成 10 大维度全方位研报。从基本面到技术面，从估值模型到风险提示，帮你建立系统的分析框架。

  • 10 大分析模块，覆盖财务、估值、技术、行业
  • 5 种经典估值模型（DCF、DDM、PEG 等）
  • 实时数据 + AI 深度解读，不是简单的数据堆砌
  • 流式输出，边生成边阅读
功能二

持仓追踪 + 巴菲特来信

导入你的持仓，每日收到一封 AI 生成的「巴菲特来信」。用价值投资的视角审视你的持仓，给出理性的分析和建议。

  • 券商截图一键导入持仓，VLM 智能识别
  • 每日市场温度 + 个股诊断 + 操作建议
  • 巴菲特人格化写作，严肃又不失温度
  • 历史来信存档，随时回顾
功能三

交易诊断

买入还是卖出？先问问 AI。输入你的交易意图，从价值、仓位、时机、市场等 6 个维度给出全面诊断，帮你做出更理性的决策。

  • 6 维度诊断：价值、仓位、时机、市场、结论、风险
  • 结合你的持仓全局分析，不是孤立看单只股票
  • 支持追问对话，深入探讨交易逻辑
  • 历史诊断存档，复盘交易决策

3 步开始使用

1

克隆仓库

git clone https://github.com/firyrice/zhixinglu.git
cd zhixinglu
pip install -r requirements.txt
2

配置环境

cp .env.example .env
# 编辑 .env，填入你的 LLM API Key
# 支持任何 OpenAI 兼容接口
3

启动服务

python3 run.py
# 访问 http://localhost:5001

环境要求：Python 3.8+，需要 LLM API Key（支持 OpenAI、Claude、Gemini 等兼容接口）
